Resources
  1. Read the PDF user guide and follow the easy or manual setup instructions
  2. For the full and easy install experience, download one of the resource folder zips from here
  3. Install REAPER and combine with the resource folder:
    • Windows: Download REAPER and check the "Portable Install" box when installing. Unzip and copy the contents of the downloaded resource folder into the portable install directory e.g. C:/REAPER.
    • MacOS: Download REAPER and drag REAPER64.APP into the downloaded resource folder
    • Linux: Download and unzip the REAPER zip and drag the contents of the downloaded resource folder into the REAPER subfolder
