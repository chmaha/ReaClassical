Quick Start Guide

Welcome to the ReaClassical Quick Start Guide. Use the dropdown menu above to select a section and get started with installation, recording, editing, mixing, mastering and more! This guide will walk you through each step with helpful instructions and images. For more detailed information and to get the most out of ReaClassical, please read the PDF Manual.

Installation & Update
Windows
  1. Download 64-bit or 32-bit install binary where you'd like to place the ReaClassical folder
  2. Double Click!
MacOS
  1. Open a terminal and navigate to where you'd like the ReaClassical folder to be located.
  2. Copy this command to the clipboard Command copied to clipboard!
  3. Paste into the terminal and press enter!
Linux
  1. Open a terminal and navigate to where you'd like the ReaClassical folder to be located.
  2. Copy this command to the clipboard Command copied to clipboard!
  3. Paste into the terminal and press enter!
Updating
  1. Press Shift+U to update ReaClassical.
  2. Wait for the ReaPack update process to complete then answer "Yes" to update the ReaClassical keymap and toolbar as necessary.
  3. Press Ctrl+U to update REAPER. Check here for the latest tested version.
Horizontal Workflow

Takes are recorded or imported left to right and edited in the same single folder.

  1. Press F7 in an empty project.
  2. Choose the number of tracks you need and name them.
  3. If recording, auto set recording inputs based on track names ("pair", "stereo" etc).
  4. Rename "M:" mixer tracks and press F7 to sync the names to regular tracks.
  5. Convert to a vertical workflow at any point by pressing F8.
Vertical Workflow

Takes are recorded or imported top to down in the arrange window in as many "source" folders as desired with edits compiled in the uppermost "destination" group.

  1. Press F8 in an empty project.
  2. Choose the number of tracks you need and name them.
  3. If recording, auto set recording inputs based on track names ("pair", "stereo" etc).
  4. You now have one "destination" group and six "source" groups
  5. Rename "M:" mixer tracks and press F8 to sync the names to all track folders.
Adding, Deleting and Rearranging Tracks
  1. To add a track to all groups, press Shift+T.
  2. To Remove a track from all groups, select the associated "M:" mixer track and press Ctrl+Shift+Delete.
  3. To rearrange tracks, drag one or more "M:" mixer tracks into the order you want then sync via F7 or F8.
Recording
  1. Press F7 or F8.
  2. Choose the number of tracks you need and name them.
  3. Auto set recording inputs based on track names ("pair", "stereo" etc).
  4. Press F9, and check recording levels.
  5. Press F9 to start and stop recording until finished.
  6. Quickly disable record-arm on all tracks and audition your material by hovering the mouse over the desired location and pressing A. For more details, refer to the "Auditioning" section.
Importing mono/stereo files
  1. Press F7 or F8.
  2. Choose number of tracks and name them.
  3. Decline to auto set recording inputs based on track names.
  4. Drag in files in groups, take by take
  5. Press T to prepare takes for editing
Importing Polywavs
  1. In an empty project double-click in the track panel to create one or more regular REAPER tracks.
  2. Drag in one or more polywavs.
  3. Press F10.
  4. If the first two channels of your polywav are the main stereo pair, answer "yes" to the stereo-interleaved question.
Preparing Takes
  • Press T to prepare takes ready for editing.
  • This is especially important if you inherited another project file in which child track takes don't necessarily start at the same time as the parent due to previous editing attempts.
  • You can press T whenever you like during the editing process.
  • You can also manually color item groups by selecting the parent item and pressing K. Subsequent runs of Prepare Takes function will honor these custom colors.
Navigating
  1. Find a recorded or imported take by number and optional session name by pressing Enter.
  2. Move to a time within an item or the whole project via TAB.
  3. Shuttle back and forth along the timeline item by item using Q and W.
  4. Move between markers using , (comma) and . (period/full stop).
  5. Zoom to entire project horizontally and vertically via ` (backtick) and Ctrl+`.
  6. Show and hide child tracks of folders via D ("display") and E ("ensconce").
  7. Adjust zoom of wave peaks via Ctrl+up/down arrows.
Auditioning
  1. Hover over a parent track item and press A to hear the whole mix
  2. Hover over a child track to just hear that track
  3. Press spacebar to stop playback
  4. Mute or solo mixer tracks to create a custom audition mix (click-swipe to affect multiple tracks in a row)
S-D Editing
  1. Add S-D markers via 1,2,3 & 4 shortcuts. To place source markers on lower source group material, click on the item to place the edit cursor.
  2. Press 5.
  3. Optionally use destination and source "project" markers for multi-project tab S-D editing.
  4. Optionally set S-D marker placement via mouse hover location.
Assembly Line Edits
  1. Add one destination marker (1) and both source markers (3,4). To place source markers on lower source group material, click on the item to place the edit cursor.
  2. Press F3.
  3. The destination marker is moved to the end of the edit ready for the next edit.
  4. If you accidentally move the destination marker, don't worry. The function will offer to place it back in the correct position.
Insert with Time-stretch
  1. Add S-D markers via 1,2,3 & 4 shortcuts. To place source markers on lower source group material, click on the item to place the edit cursor.
  2. Press F4.
Crossfade Editor Basics
  1. Select the right-hand item of a crossfade pair and press F.
  2. Press Z to mirror extend the edges of both items.
  3. Align green item transients with red's
  4. Place edit cursor at desired position and press X.
  5. Press F to leave the crossfade editor or press Q or W to go to previous or next crossfade.
Auditioning Inside the Crossfade Editor
  1. Hover the mouse and press A to audition a crossfade.
  2. Hover and press A in empty space to set mirrored audition in/out points.
  3. Hover and press A on the red item to only hear the left side of the fade.
  4. Hover and press A on the green item to only hear the right side of the fade.
Mixing Basics
  1. ReaClassical uses a single mixer paradiym like Merging Pyramix. All regular tracks feed into mixer tracks (with an "M:" prefix). All routing ends up at an "RCMASTER" bus.
  2. Add a hyphen (-) suffix to any "M:" mixer track to disable direct connection to RCMASTER. This is very useful for setting up mixbusses.Press F7 or F8 to set active.
Aux & Submixes
  1. Press @ to add an aux send (i.e. reverb) or # to add a submix track.
  2. You can add hyphens (-) to aux or submix tracks too to create extremely customized routing. Press F7 or F8 to set active.
  3. Click-drag from stripes on "M:" mixer track to the special track you want to connect to.
  4. Likewise, in the case of multiple layers of mixbusses, click-drag from one to another.
Room Tone & Reference Tracks
  1. Press ~ (tilde) to set up a roomtone track and drag in some generated room tone (see "mastering" section for more details).
  2. Press ' (single quote) to set up a reference track and drag in some audio. This is outside the RCMASTER structure enabling quick A/Bing of your material against others.
  3. You have the ability to set the reference track to be an overdub guide (see PDF manual for details).
Automation
  1. Enter Mastering Mode by setting in ReaClassical Preferences (F5).
  2. Engage Automation Mode by Ctrl+I
  3. Change a fader or any other mixer control or FX parameter
  4. Drag a time selection and press I to only affect that area.
  5. Alternatively place the edit cursor and press I to affect everything to the right of the cursor.
  6. Press Ctrl+I to exit automation mode.
Mastering an Album
  1. Press Ctrl+T to remove all names from items on the destination parent track.
  2. Double-click on, or select and press F2, only items that should be treated as album track starts.
  3. Press Y to start the CD/Album wizard.
  4. Enter metadata, optionally ISRC etc.
  5. To export an album report and CUE file, press C.
Room Tone
  1. If you have some generated room tone the length of your album, pressing Y will also create seamless automation fade in and outs to dovetail with audio.
  2. To re-order tracks, select an item that starts a CD/album track and press Ctrl+left or right to move.
  3. To re-position all CD tracks to have the same gaps, press Ctrl+Y and enter a number in seconds.
Re-ordering and Re-positioning Album Tracks
  1. If you have some generated room tone the length of your album, pressing Y will also create seamless automation fade in and outs to dovetail with audio.
  2. To re-order tracks, select an item that starts a CD/album track and press Ctrl+left or right to move.
  3. To re-position all CD tracks to have the same gaps, press Ctrl+Y and enter a number in seconds.
Rendering
  1. Once your album is ready to export, press R.
  2. Select a ReaClassical preset from the dropdown menu under "All Settings".
  3. Optionally add normalizing and/or limiting
  4. Press "Dry Run" or "Render"
ReaClassical Plugins

ReaClassical ships with a number of "mastering" grade plugins. For more detail, please read Appendix C of the PDF Manual.

  • Two EQs: RCChannelEQ, RCMasteringEQ
  • Two compressors: RCCompressor & RCParallelK Compressor
  • A true peak limiter: RCLimiter
  • Utilities: RCGain, RCTime&PhaseTool (for accurately timing spot microphones) and RCMidSide (for encoding and decoding Mid-Side signals)
  • 24-bit and 16-bit Dither via RCDither
  • RCDeEss & RCDeEss Advanced
  • RCConsole Series for adding an analog touch
  • Final polish via RCInflator Ultimate (Sonnox clone) and MagicFairyDust
  • Plus third-party offerings including ReEQ (a Pro-Q clone), ReSpectrum, and ReaPlugs, REAPER's own fantastic plugin set
